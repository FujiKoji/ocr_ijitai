# OCR_IJITAI
OCR_IJITAI is optical character recognition engine to recognize "ijitai".
The "ijitai" has the same meaning as "高" and "髙" but is a different form of the Chinese character.

# Requirement
* python 3.6.0
* pillow 9.2.0
* torch 1.12.1
* scikit-learn 1.1.1
* opencv-python 4.6.0.66
